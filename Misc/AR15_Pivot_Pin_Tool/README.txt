# Summary

This tool makes detent work faster and effortless:

Insert tool into pivot pin holes from the left side.
Align hole in tool with detent hole.
Insert spring and detent through tool hole into receiver.
Compress detent in recess using 3/32" punch and rotate tool.
Push out tool with pivot pin and rotate until detent is in groove of pivot pin.
